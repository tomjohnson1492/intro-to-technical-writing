What Technical Writers Do

  • Create documentation
  • Contribute UI text
  • Make information Visual
  • Create strategies for re-use
  • Set up authoring and publishing tools.

Major Challenges

  • Understand how a product works
  • Get information from engineers
  • Work in isolation
  • Meet regulation standards

Job Titles

  • Information developer
  • Knowledge engineer
  • Content strategist
  • Technical communicator
  • User assistance professional

  • English major
  • Software engineer
  • Self-learning techophile
  • Library scientist
  • Linguistics expert
  • Hobby filmmaker

Ideal Training

  • Double major in computer science and English
  • Any engineering discipline, but esp. software
  • Graphics and digital media
  • Computer programming
  • Biology and life sciences

Valuable Attributes

  • Writing skills
  • Good with computers
  • Patience when troubleshooting
  • Courage to ask questions
  • Okay with low-value status in org
